This is for the macro to build in rust-analyzer / vscode and not hang for a long time when there is no cache. If the user is offline, the timeout ends and macro makes a compile error that says "No cache found for the endpoint and failed to fetch the spec from the endpoint. Please make sure you are online and the endpoint is correct, or enable update_cache feature to fetch the spec with normal timeout." This is for the user to understand why the macro is not working and how to fix it.
